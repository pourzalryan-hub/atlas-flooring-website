## WIX STUDIO IMPLEMENTATION NOTES

### Form Setup
1. Wix Forms → Custom Form → add all fields listed above
2. Connect to email notification → your business email
3. Add automated confirmation email to the user (set in Form Settings → Submissions → Email Confirmation)
4. Optional: Connect to Wix CRM or export to spreadsheet for lead tracking

### Wix Bookings Integration (Optional but Recommended)
1. Install Wix Bookings app
2. Create two service types:
   - "Showroom Consultation" — in-person, 60 min
   - "In-Home Consultation" — your location, 60 min, service area defined
3. Embed booking widget or link `Get a Free Consultation` buttons to Wix Bookings URL

### Google Maps Embed
1. Go to Google Maps → search your address → Share → Embed a map
2. Copy the iframe code
3. In Wix Studio → Add → Embed → Custom HTML → paste the code
4. Set height to 450px, width 100%
